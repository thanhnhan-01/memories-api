# Memories API

A RESTful backend service for a personal Memories application.
Users can create, edit, organize, and search memories with image attachments and authentication support.

## Tech Stack
- Java 21
- Spring Boot 3
- Spring Security
- JWT Authentication
- PostgreSQL 16
- Maven
- Docker
